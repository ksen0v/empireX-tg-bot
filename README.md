## EmpireX Bot



#INSTALL (FOR WINDOWS)

- Create virtual environment:
 
  ```sh
  python -m venv venv
  ```

- Run virtual environment:
 
  ```sh
  ./venv/Scripts/activate
  ```

- Install requirements:
 
  ```sh
  pip intall -r requirements.txt
  ```

- Insert your bot_token and admin_id in config.json
- Run bot:

  ```sh
  python main.py
  ```
